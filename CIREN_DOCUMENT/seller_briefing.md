## 2. What is CIREN?

CIREN is an **industrial IoT monitoring system** that lets factories and facilities track real-time data from sensors — temperature, humidity, vibration, voltage — on a live dashboard, accessible from any browser or smartphone.

**The key difference from competitors:**
Most industrial IoT systems require weeks of installation, custom software, and dedicated IT staff. CIREN works in minutes. You physically plug in a sensor, and it appears on the dashboard automatically. No configuration. No code. No IT department.

## 4. The 3 Things That Make CIREN Different

### ① Truly plug-and-play
Not marketing language — it literally works by plugging in a sensor cable. The system detects what sensor type it is and starts displaying data. You can swap sensors while the system is running.

### ② Never goes offline
CIREN has **WiFi + LTE-M (cellular) dual connection**. If the factory WiFi drops, it automatically switches to cellular and keeps sending data. For 24/7 monitoring this is critical — most competitors are WiFi-only.

### ③ Multiple sensors, one dashboard
Temperature, humidity, vibration, voltage — mix and match up to 8 sensors per controller. All appear on the same screen. Useful for factories that need to monitor different parameters in different zones.

```

4. "Now let me show what happens with vibration."

5. "This works the same way at any of your factory locations,
   visible from your phone or any browser."
```

**Key moment:** The instant the sensor card appears after plugging in. Let the visitor watch in silence for 3 seconds. That pause sells it.

## 4. CIRENが選ばれる3つの理由

### ① 本当のプラグ＆プレイ
マーケティング用語ではありません。センサーケーブルを挿すだけで自動認識し、即座にデータ表示が始まります。稼働中でもセンサーの交換・追加が可能です。

### ② 通信が途切れない設計
**WiFi＋LTE-M（セルラー）デュアル接続**を標準搭載。工場のWiFiが切断されても自動でLTE-Mに切り替わり、データ送信を継続します。24時間監視が求められる現場に最適。WiFiのみ対応の競合製品との大きな差別化ポイントです。

### ③ 複数センサー・一元管理
温度・湿度・振動・電圧など、1台のコントローラーに最大8個のセンサーを接続可能。用途に応じて自由に組み合わせ、すべて同じ画面で確認できます。

## 6. デモの流れ（約2分）

```
1.「実際に見ていただきましょう。」
   → モニターに流れているビデオを指す。

2.「今はセンサーが何もつながっていません。」
   → DHT20（温湿度センサー）を接続する。

3.「ダッシュボードをご覧ください。」

4.「次に振動センサーです。」
   → MPU6050（IMU）センサーを接続する。

5.「これと同じ画面が、御社のどの拠点からでも、
   スマートフォンでご確認いただけます。」
```

**ポイント：** センサーを挿した直後にカードが表示される瞬間が最大の見せ場です。3秒間、無言で画面を見ていただいてください。その沈黙が一番の説得力になります。
